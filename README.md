# Refraction-Shader (Refraction Raytracer)
### 物理ベースの屈折率(IOR)を用いた光線追跡実装

## 概要
光学的な屈折現象を数学的に記述し、**C++でスクラッチ実装**したレイトレーシング・プログラムです。外部ライブラリを使用せず、**スネルの法則**に基づいた光線の挙動をシミュレートし、物理的に正しい透過・屈折の描画を行います。


## 技術的特徴
- **物理ベースの屈折演算**: 物質固有の**屈折率（IOR: Index of Refraction）**に基づき、スネルの法則をベクトル演算で実装。**全反射（Total Internal Reflection）**の判定ロジックも構築しています。
- **幾何学的な交差判定**: 光線（Ray）と球体（Sphere）の**交差判定アルゴリズム**を二次方程式の解法から自作。
- **透視投影（Perspective Projection）**: ピンホールカメラモデルをベースに、画角に応じた**自然な空間の歪み**を再現する透視投影を実装。
- **シェーダーへの拡張性**: 本プログラムのロジックは**HLSL/GLSL等のシェーダー言語**へ直接移植可能な設計となっています。

## 実装の詳細
- **言語**: **C++** (外部ライブラリ非依存)
- **主要機能**: 
  - `refract` 関数による**物理的に正確な屈折方向**の算出
  - 球体との衝突点における**表面法線ベクトル**の計算
  - PPM形式での画像出力
