##物理ベースの屈折率(IOR)を用いた光線追跡

##概要
光学的な屈折現象を数学的に記述し、スクラッチで実装したレイトレーシング・プログラムです。外部ライブラリに頼らず、スネルの法則に基づいた光線の挙動をシミュレートし、物理的に正しい透過・屈折の描画を行います。

##技術的特徴
物理ベースの屈折演算: 物質固有の屈折率（IOR: Index of Refraction）に基づき、スネルの法則をベクトル演算で実装。全反射（Total Internal Reflection）の判定もアルゴリズムに含んでいます。

スクラッチ実装の交差判定: 光線（Ray）と球体（Sphere）の幾何学的な交差判定を数式から実装。

透視投影（Perspective Projection）: ピンホールカメラモデルに基づく透視投影を実装し、画角に応じた自然な歪みを再現。

拡張性: ロジックはHLSL/GLSL等のシェーダー言語へ移植可能な設計となっており、空間再現ディスプレイ等のリアルタイム描画への応用を想定しています。
